Data Types in Python

Data Type Description Examples
int Integer value 10, -5, 0
float Floating-point value 3.14, -0.5, 1.0
str String of characters "Hello", 'World', "123"
bool Boolean value (True/False) True, False
list Ordered collection of items [1, 2, 3], ['apple', 'banana', 'orange']
tuple Immutable ordered collection of items (1, 2, 3), ('apple', 'banana', 'orange')
dict Key-value pairs {'name': 'John', 'age': 25}
set Unordered collection of unique items {1, 2, 3}, {'apple', 'banana', 'orange'}
